# E-Commerce-app-using-MERN
[Link to the website](https://watchverse.shop/)
## Features:
* User & Admin Authentication
* Payment gateways integrated: Razorpay
* Cash On Delivery
* Offers and Coupons
* Sales Reports
* User profile
* Image Zooming
* Multiple Address Management
* User Block/Unblock
* Download Invoice
* Wallet and wallet history
* Chart & Graph reports on Admin side
* Form Validation
* Product & Category Management in Admin side
* Load balancer

## Technologies used:
* Node js
* Express js
* MongoDB
* Bootstrap
* Nginx

## Services used:
* AWS
